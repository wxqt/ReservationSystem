# 校园机房预约系统

## 说明

这是一个学习C++时教程上的一个小程序 是再命令行界面下实现的
我用Qt实现了一个简单的图形程序
这是最初的简单版
这一版用的简单的文本文件存储用户密码等东西

以后会用二进制 单机数据库 远程数据库再重新实现
再把学到的继承 多态全用上

这个程序学C++还是不错了

## PS:

由于是简单版 管理用户名和密码默认都是admin
也没有提供修改管理员密码方案

## 基本流程

### 管理员

* 添加学生
* 添加老师
* 添加机房

### 学生

* 添加预约
* 取消预约
* 查看自身和全部预约

### 老师

* 审核通过功能
